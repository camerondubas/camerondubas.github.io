Change Heroes Fundraising

Online Fundraising Platform

Change Heroes is an Friend-to-Friend Fundraising Platform that enables individuals to channel their friends and raise money for a tangible charitable cause. To date, Change Heroes has raised over $3.5M for charity. This project is multiple Angular Apps served bya Node Server with a Java REST API. Some technical highlights include User Authentication, Payment Processing, and Video Recording.

Change Heroes Beta

Rebuild of the Change Heroes Platform

This is a SaaS version of the original Change Heroes Platform. It is a technical improvement in every aspect. It is built on an Angular Front-end and a Node REST API with PostgreSQL and a Database with scalability in mind. Some major improvements include lazy-loading of assets, mobile-first development approach, and a modern Webpack workflow deployed to AWS via Docker Containers.

Node.js RSS API

Simple CORS-Enabled API to convert and RSS Feed into usable JSON

I was working a side project and wanted to pull an RSS feed. However I kept running into Cross-Origin Resource Sharing (CORS) issues. The project was small and didn't warrant a backend just to accomplish this small task. This RSS API solves that issue, simply send the URL of an RSS feed and the contents will be returned in JSON.
